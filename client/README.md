# Syn By Design Client

## Get Started

1. `cp dev.env .env`
2. `npm ci`

## Run locally for development

1. `npm run build`
2. `npm run dev`

## Run app for production
1. `npm run build`
2. `npm start`

## Testing

`npm test`

## Deploy via `now`

`npm run deploy`

### Deploy to production

`npm run deploy:prod`
